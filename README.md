# markdown2HTML
Provides a markdown editor that convert the text to html to show the final aspect.

# Requirements

1- install PyQt5

# Features

- [x] Create new text file.
- [x] Edit a text file.
- [x] Save a text file.
- [x] Open a text file.
- [ ] Show HTML.
- [ ] Save HTML.

# Non Functional Features

- [x] Header menu.
- [ ] Documentation generated with code.
- [ ] Translation to other languajes.
